How to add the necessary permission

You should have an "Administrator" role with "Download files" permission to load preview.

Team settings → User groups

Create new group button

Permissions tab

Check "Administrator role"

In the "Administrative permissions" section uncheck everything except "Download files"

"Save permissions" to apply changes
